# Airbnb Clone Web Application

## Tech stack

1. Nextjs
2. tailwindcss
3. next js api for data

### How to run app

1. npm install from root
2. npm run dev

### Features

1. search
2. Animation
3. Light and Dark Mode
4. Map
5. bar-of-progress
6. Add as favorite
7. Sign in (google api fix needed)
8. payment (api fix needed)
